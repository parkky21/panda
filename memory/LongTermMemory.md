# Long Term Memory Index

This is the root index of Panda's long-term memory graph. It contains links to modular memory files about the user's life, preferences, and history.

## Memory Graph Tree
* [User Profile](user_profile.md) - Immediate context about who the user is as a person.
* [Childhood](nodes/childhood.md) - Memories and stories from the user's youth.
* [Hobbies & Interests](nodes/hobbies.md) - Things the user enjoys doing.
* [Career & Projects](nodes/career.md) - Professional history, programming projects, and goals.
* [Childhood memories and stories](nodes/childhood.md)
